Privacy Policy

Last updated: December 13, 2024

TL;DR: This extension does NOT collect, store, transmit, or share any of your data. Everything happens locally in your browser. Zero data collection. Zero tracking. Zero external servers.

Introduction

LinkedIn AI Post Detector ("the Extension") is committed to protecting your privacy. This privacy policy explains how the Extension handles data and what permissions it requires to function.

Data Collection

The Extension does NOT collect any user data.

We do not collect, store, transmit, or share:

  • Personal information
  • LinkedIn account credentials
  • LinkedIn posts or content you analyze
  • Browsing history
  • Usage statistics or analytics
  • Any other form of user data

How It Works

When you use the Extension to analyze a LinkedIn post:

  1. You manually select a post using the element picker
  2. Text is extracted from the selected element in your browser
  3. Analysis is performed entirely client-side using JavaScript
  4. Results are displayed only to you in the popup
  5. No data is sent to external servers
  6. Nothing is stored after you close the popup (unless you enable history in settings)

Local Storage

The Extension uses Chrome's local storage only to save your preferences and settings:

  • Settings: Your detection sensitivity, enabled methods, and display preferences
  • Optional History: If you enable "Keep Analysis History" in settings, the last 10 analyses are stored locally on your device

This data never leaves your device and can be cleared at any time through the settings page or by uninstalling the Extension.

Permissions Explained

The Extension requests the following minimal permissions:

activeTab

Allows the Extension to interact with the currently active LinkedIn tab when you click the extension icon. This is required to inject the element picker and extract text from the post you select.

storage

Allows the Extension to save your settings and preferences locally in your browser. No data is synced to external servers.

scripting

Allows the Extension to inject the element picker interface and analysis scripts into LinkedIn pages. This only happens when you activate the picker.

host_permissions (linkedin.com)

Limits the Extension to only work on LinkedIn.com. The Extension cannot access or interact with any other websites.

Third-Party Services

The Extension does not use any third-party services, APIs, or analytics tools.

All functionality is self-contained within the Extension code. There are no external API calls, no tracking pixels, no analytics libraries, and no connections to external servers.

LinkedIn Terms of Service

The Extension is designed to respect LinkedIn's Terms of Service:

  • User-initiated only: No automatic scraping or data collection
  • Publicly visible content: Only analyzes content already loaded and visible to you
  • No bulk operations: Analyzes one post at a time, manually selected by you
  • Equivalent to manual actions: Functionally similar to manually copying text and analyzing it yourself

Data Security

Since the Extension does not collect, transmit, or store any data externally, there are no data security risks associated with external servers or databases.

Your settings and optional history are stored locally using Chrome's secure storage API and are protected by your browser's security measures.

Children's Privacy

The Extension does not knowingly collect any information from anyone, including children under 13. The Extension does not collect data from any users.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last updated" date at the top of this page. Continued use of the Extension after changes indicates acceptance of the updated policy.

Open Source

The Extension's source code is available for review. You can verify that the Extension does not collect or transmit data by reviewing the code yourself.

Your Rights

Since the Extension does not collect any personal data, there is no personal data to access, modify, or delete. You can:

  • Clear your settings by clicking "Reset to Defaults" in the settings page
  • Clear your analysis history (if enabled) through the settings page
  • Remove all Extension data by uninstalling the Extension

Disclaimer

This Extension provides estimations based on writing patterns and is not a definitive method for detecting AI-generated content. Results should be used for personal reference only and not as conclusive proof.

The Extension is for personal, educational use only. Use at your own discretion and risk.
